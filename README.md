O diretório Flappy contém o jogo editado e desenvolvido para o projeto, incluindo tanto a versão inicial quanto a versão com testes de IA.

A versão inicial apresenta a mecânica básica do jogo, enquanto a versão com IA implementa algoritmos para análise e tomada de decisão automatizada. O objetivo é explorar a aplicação da inteligência artificial no aprimoramento do desempenho e na adaptação do jogo.
